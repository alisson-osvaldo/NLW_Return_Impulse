O que é: https://tailwindcss.com

Ele é uma ferramenta de utilitarios css, ou seja varias classes que adicionam estilos nos nossos elementos html

tipo um css in js
 
css direto no js ou direto na tag HTML

HTML, JS e CSS tudo em um so lugar

Dessa forma o componente fica bem mais tranquilo de trabalhar 

https://tailwindcss.com/docs/installation

O proprio reseta todos os estilos da nossa aplicação
